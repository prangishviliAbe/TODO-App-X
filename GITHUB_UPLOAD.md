# ­Ъџђ GitHub-рЃќрЃћ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

## ­ЪЊІ рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў

### 1. Git-рЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ
1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ: https://git-scm.com/download/win
2. рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ Windows-рЃўрЃАрЃЌрЃЋрЃўрЃА
3. рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ (рЃДрЃЋрЃћрЃџрЃљ рЃюрЃљрЃЉрЃўрЃ»рЃќрЃћ "Next" рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ)
4. рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў

### 2. GitHub-рЃќрЃћ Repository-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ: https://github.com
2. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў (рЃЌрЃБ рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ)
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "New repository"
4. рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ рЃАрЃљрЃ«рЃћрЃџрЃў: `todo-app`
5. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ "Public"
6. **рЃљрЃа рЃЊрЃљрЃљрЃГрЃўрЃарЃЮрЃЌ** "Add a README file"
7. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "Create repository"

### 3. рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ
PowerShell-рЃерЃў рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў:

```bash
# Git-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ
git config --global user.name "рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃ«рЃћрЃџрЃў"

# Repository-рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ

# рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

# рЃърЃўрЃарЃЋрЃћрЃџрЃў commit

# GitHub-рЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ

git branch -M main
git push -u origin main
```

### 4. рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџрЃў рЃњрЃќрЃљ - GitHub Desktop
1. рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ: https://desktop.github.com/
2. рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ
3. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў repository
4. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃцрЃљрЃўрЃџрЃў
5. рЃњрЃљрЃЏрЃЮрЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЌ

## ­ЪЊЂ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ
- РюЁ `index.html` - рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў HTML рЃцрЃљрЃўрЃџрЃў
- РюЁ `styles.css` - рЃДрЃЋрЃћрЃџрЃљ рЃАрЃбрЃўрЃџрЃў рЃЊрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ
- РюЁ `script.js` - JavaScript рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў
- РюЁ `README.md` - рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ
- РюЁ `README_EN.md` - рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ
- РюЁ `.gitignore` - Git ignore рЃцрЃљрЃўрЃџрЃў
